Ableton Remote Script for LinnStrument 128 and 200

This script adds clip launcher and a mixer functionality to your LinnStrument while using it in Ableton Live.
It's a custom remote script focused on live looping in session view.
Study the included PDF for all mapped functions of the remote Script.

1. Unzip Remote Script
2. Copy Folder "Linnstrument" to your Ableton User Libary / Remote Scripts
3. On Linnstrument: Left Split: CC-Fader (1-8), 1 Row
4. Open Ableton
5. Ableton Preferences, Remote Scripts, choose "Linnstrument" and "Linnstrument Midi" for Input / Output
6. Switch betwenn "User Mode" and "Session Mode" with Pad 8, Row 1 on your Linnstrument
8. If you have a Linnstrument 200: push a pad on row 17-25 after launch of Ableton
9. Bug: Please make shure to switch back to "Notes Mode" before changing Ableton Sets (or your LinnStrument will freeze). Simply unplug and reconnect your LinnStrument if that occurs. I'm trying to resolve this issue..
10. Enjoy your LinnStrument with a lot of APC 40 mk2 & Push functionality!
